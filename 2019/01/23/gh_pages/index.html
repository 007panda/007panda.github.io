github pages + hexo搭建个人网页

[TOC]

概述

首先大致了解一下github_pages和hexo：

github-pages是github的一个服务，可以显示一个自己的网页

hexo用来创建一个网站。它基于node.js，需要先下载node.js的环境。之后使用npm安装hexo，修改配置，安装插件即可使用它在github上显示页面。

一、环境配置

  1. 安装node.js及npm、git

  2. 配置github上ssh密钥对,使得本地可以ssh连接github

    可以使用下面的命令即可查看是否已经配置成功

二、github_pages的创建

在github中创建一个新的资源库，将其命名为your_github_name.github.io即可。创建后，即可在settings界面下看到github_page处显示相关信息

三、安装hexo

在要安装的文件夹下使用 npm install hexo(-cli) -g

-g 参数意思为global——全局安装，会安装到npmrc文件中配置的目录中，可以直接在命令行中使用

关于-g、–save、 –save-dev等参数的使用说明：

https://blog.csdn.net/u011565547/article/details/79646075

https://blog.51cto.com/9161018/2322229

四、hexo初始化自定义目录

在一个目录中，执行hexo init命令，即会在这个目录中克隆相应的文件。
