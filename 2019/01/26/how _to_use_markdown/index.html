Markdown

What Is Markdown?

Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。

markdown 正如它的名字一样，是一种标记语言。与之类似的有html(HyperText Markup Language),xml(Extensible Markup Language)等。它不同于其他标记语言的特点是，简单明了，编写方便，并且它设计之初的宗旨即是简单——易读易写。虽然它很简单，但是仍可以展现丰富的效果，因此有着很多的使用场景。

What It Can Do?

Markdown的语法简洁明了、学习容易，而且功能比纯文本更强，因此有很多人用它写博客。

用于编写说明文档，并且以“README.MD”的文件名保存在软件的目录下面。

除此之外，由于我们有了RStudio这样的神级编辑器，我们还可以快速将Markdown转化为演讲PPT、Word产品文档、LaTex论文甚至是用非常少量的代码完成最小可用原型。在数据科学领域，Markdown已经广泛使用，极大地推进了动态可重复性研究的历史进程。

我自己最开始使用markdown，是用来做笔记，它也确实很适合用来做这个。

How to use it?

标题

使用
#title1# ##title2## ###title3### ...

即可产生如下的效果

title1

title2

title3

title4

段落和换行

每当你想要开始新的段落时，只需要插入一行空白行即可。

比如：

frist line 

second line

展示效果：

frist line

second line

分割线

在一行中使用三个及以上星号*、减号-或者底线_来建立一个分割线，其中可以有间隔，但是行内不能存在其他东西。

引用

使用“>”符号来引用

使用：

>this is a quote.

效果：

this is a qoute.

代码

直接缩进4个空格即可开始一个代码段

在行内使用时，在要标记的部分前后加上 ` 即可

强调

使用星号 * 和 下划线 _ 作为标记强调字词的符号

链接和图片

链接

  • 简单的使用方式：
    [这是描述文字](--url-- "这是鼠标放置时显示的文字")

实际效果：   

[百度](https://www.baidu.com "点击即可开打百度主页")

  • 特殊点的，你可以把()内的内容用一个标号代替，在其他地方将这个标号的详细内容单独定义即可:

    [这是描述文字][url]

[url]: --url-- "这是鼠标放置时显示的文字"

    实际效果：

    [百度][baidu]

    [baidu]: https://www.baidu.com “点击即可打开百度搜索主页”

图片

和链接类似，只需要在前面加上一个！即可
